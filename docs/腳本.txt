# Role

你是一位資深的電腦視覺與深度學習工程師，專精於 YOLO 系列模型的遷移學習（Transfer Learning）與影片串流邏輯開發。



# Context

- 效能表現：mAP50=0.983, Precision=0.965, Recall=0.983。

- 下一階段目標：以 `train.pt` 為基底進行微調，並開發「虛擬圍籬（Bounding Box Trigger）」影片擷取模組。



# Task 1: Fine-tuning Script (微調策略)

請撰寫一個 Python 腳本，使用 `ultralytics` 載入 `best.pt` 進行訓練，要求如下：

1. **權重繼承**：明確載入自定義權重檔案，而非官方預訓練權重。

2. **凍結層設定**：加入 `freeze=10` 參數，凍結 Backbone 前 10 層以保留基礎特徵，僅對後端偵測頭進行微調，適應新場景。

3. **超參數調優**：

   - 採用更低的起始學習率 `lr0=0.0001` 以維持穩定。

   - 啟用 `cos_lr=True` 進行餘弦退火調整。

   - 加入 `mosaic=0.5` 與 `mixup=0.1` 提升模型對重疊目標的魯棒性。

4. **早停機制**：設定 `patience=20`，若連續 20 個 Epoch 無改善則停止訓練。



# Task 2: Virtual Fence Video Trimming (自動化擷取邏輯)

開發一個處理輸入影片（.mp4）的模組，邏輯如下：

1. **觸發區域定義**：允許透過座標設定一個「虛擬感應框（Trigger ROI）」。

2. **狀態機判定 (State Machine)**：

   - **進入 (Enter)**：當乳牛偵測框（BBox）中心點進入感應框時，標記「開始時間點」。

   - **持續 (Tracking)**：在感應框內持續追蹤。

   - **離開 (Exit)**：當中心點離開感應框或偵測消失時，標記「結束時間點」。

3. **影片輸出**：根據「開始與結束」的時間區間，自動剪輯原始影片並儲存為獨立的 `.mp4` 檔案。
